# Segurança e Boas Práticas

Este projeto foi pensado para minimizar riscos ao usuário.

## Princípios
- Não clona jogos
- Não modifica APKs
- Não injeta código em jogos
- Não usa root

## Funcionamento seguro
- Usa apenas eventos normais do sistema Android
- Respeita limites de frequência de toque
- Evita padrões artificiais

## Aviso
Nenhum método é 100% seguro contra banimentos.
O uso é por conta e risco do usuário.

Projeto educacional e experimental.
