# Plano do Projeto

## Objetivo
Criar um input mapper para Android sem root,
sem clonagem e com foco em menor risco de ban.

## Etapas
1. Estrutura do projeto
2. App Android básico
3. Integração Shizuku
4. Overlay flutuante
5. Modo FPS seguro

## Regras fixas
- Nunca usar PC
- Nunca usar clonagem
- Nunca usar root
